# webppl-linesim

Checkout [main.js](./main.js)!

## Getting started

1. Install nodejs v15.5.1, e.g., using [nvm](https://github.com/nvm-sh/nvm)
2. Run `npm install`
3. Run `./node_modules/.bin/webppl main.js`
